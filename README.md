# WIN3XLDR
WIN3XLDR is Windows 3.x Loader for MS-DOS 7.1/8.0. It is a stealth unloadable TSR. It is an adapter. It allows you to load unmodified Windows 3.x on unmodified MS-DOS 7.1/8.0. By using WIN3XLDR, you do not need to patch IO.SYS, not in-file, nor in-memory.

See WIN3XLDR.TXT and comments in WIN3XLDR.ASM for details.

Run

```
WIN3XLDR.COM /?
```

to see the usage.
